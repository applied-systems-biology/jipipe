    Annotation

    This document describes the properties of the JSON API object. You also can access the document in JSON Schema format: annotation.schema.json

    An annotation

    name: string

    Name of the annotation

    value: string

    Value of the annotation

    Example

    {
    "name" : "#Dataset",
    "value": "Experiment 1"
}
